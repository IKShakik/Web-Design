CSS Margin

CSS margin properties generate:

  • white space around the element.
  • white space outside the border.

Margin - Individual Sides:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

All the margin properties can have the following values:

  • auto - browser calculates the margin
  • length - specifies a margin in px, pt, cm, etc.
  • % - specifies a margin in % of the width of the containing element
  • inherit - specifies that the margin should be inherited from the parent element
  • Negative values are allowed

Margin - Shorthand Property Syntex:

margin: 100px 150px 100px 80px;

Auto is used:

  • to horizontally center the element within its container.
  • to specify the width
  • to split the remaining space equally between the left and right margins.

Inherit Value:

inherited from the parent element.

Margin Collapse

Only Top and bottom margins can be collapsed into a single margin.

CSS Padding

CSS padding properties generate:

  • white space around the content.
  • white space inside the border.
  • clears an area around the content (inside the border) of an element.

Padding - Individual Sides

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

All the padding properties can have the following values:

  • length - specifies a padding in px, pt, cm, etc.
  • % - specifies a padding in % of the width of the containing element
  • inherit - specifies that the padding should be inherited from the parent element

Padding - Shorthand Property Syntex:

padding: 100px 150px 100px 80px;
